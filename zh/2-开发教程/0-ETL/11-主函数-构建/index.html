    д»ҘдёӢжҳҜеҜ№ gc-qa-rag/ragapp/etl_index.py ж–Үд»¶зҡ„иҜҰз»Ҷ Markdown ж–ҮжЎЈиҜҙжҳҺпјҡ

    ж–ҮжЎЈпјҡetl_index.py д»Јз ҒиҜҰи§Ј

    ж–Үд»¶жҰӮиҝ°

    жң¬ж–Үд»¶жҳҜж•ҙдёӘ RAGпјҲRetrieval-Augmented Generationпјүж•°жҚ®еӨ„зҗҶжөҒзЁӢдёӯзҡ„ ETLпјҲExtract-Transform-Loadпјүи°ғеәҰдёҺе…ҘеҸЈжЁЎеқ—гҖӮе®ғиҙҹиҙЈж №жҚ®дёҚеҗҢзҡ„ж–ҮжЎЈзұ»еһӢпјҲеҰӮдә§е“Ғж–ҮжЎЈгҖҒи®әеқӣй—®зӯ”гҖҒи®әеқӣж•ҷзЁӢпјүе’Ңдә§е“ҒзәҝпјҢиҮӘеҠЁеҢ–ең°жү№йҮҸеӨ„зҗҶеҺҹе§Ӣж•°жҚ®ж–Үд»¶пјҢз”ҹжҲҗз”ЁдәҺеҗҺз»ӯжЈҖзҙўе’ҢеөҢе…Ҙзҡ„з»“жһ„еҢ–ж•°жҚ®гҖӮиҜҘжЁЎеқ—йҖҡиҝҮеӨҡзәҝзЁӢе№¶еҸ‘гҖҒзҒөжҙ»зҡ„й…ҚзҪ®е’Ңйқўеҗ‘еҜ№иұЎзҡ„и®ҫи®ЎпјҢжһҒеӨ§жҸҗеҚҮдәҶеӨ§и§„жЁЎж•°жҚ®еӨ„зҗҶзҡ„ж•ҲзҺҮе’ҢеҸҜз»ҙжҠӨжҖ§гҖӮ

    дё»иҰҒз»“жһ„дёҺж ёеҝғжөҒзЁӢ

    1. й…ҚзҪ®дёҺж—Ҙеҝ—

    • йҖҡиҝҮ setup_logging() еҲқе§ӢеҢ–ж—Ҙеҝ—зі»з»ҹпјҢдҫҝдәҺеҗҺз»ӯжөҒзЁӢзҡ„зӣ‘жҺ§дёҺй—®йўҳиҝҪиёӘгҖӮ
    • ProcessingConfig ж•°жҚ®зұ»е°ҒиЈ…дәҶеӨ„зҗҶжөҒзЁӢжүҖйңҖзҡ„ж ёеҝғеҸӮж•°пјҢеҢ…жӢ¬ж №и·Ҝеҫ„гҖҒдә§е“ҒеҗҚгҖҒе№¶еҸ‘зәҝзЁӢж•°е’ҢеӨ„зҗҶжЁЎејҸпјҲеҰӮвҖңfullвҖқжҲ–вҖңnoneвҖқпјүгҖӮ

    2. жҠҪиұЎеҹәзұ»дёҺеӨҡжҖҒеӨ„зҗҶ

    • BaseDocumentProcessor жҳҜжүҖжңүж–ҮжЎЈеӨ„зҗҶеҷЁзҡ„еҹәзұ»пјҢе®ҡд№үдәҶж–ҮжЎЈеӨ„зҗҶзҡ„йҖҡз”ЁжөҒзЁӢе’ҢжҺҘеҸЈпјҢеҢ…жӢ¬иҺ·еҸ–жәҗ/зӣ®ж Үи·Ҝеҫ„гҖҒиҺ·еҸ–еӨ„зҗҶеҮҪж•°гҖҒиҺ·еҸ–еҫ…еӨ„зҗҶж–Үд»¶еҲ—иЎЁгҖҒе№¶еҸ‘еӨ„зҗҶзӯүгҖӮ
    • иҜҘеҹәзұ»йҖҡиҝҮжҠҪиұЎж–№жі•е’ҢеұһжҖ§пјҢиҰҒжұӮеӯҗзұ»е®һзҺ°е…·дҪ“зҡ„ж–ҮжЎЈзұ»еһӢйҖ»иҫ‘пјҢдҝқиҜҒдәҶд»Јз Ғзҡ„еҸҜжү©еұ•жҖ§е’ҢеӨҚз”ЁжҖ§гҖӮ

    3. е…·дҪ“ж–ҮжЎЈеӨ„зҗҶеҷЁ

    • DocProcessorгҖҒForumQAProcessorгҖҒForumTutorialProcessor еҲҶеҲ«еҜ№еә”дә§е“Ғж–ҮжЎЈгҖҒи®әеқӣй—®зӯ”гҖҒи®әеқӣж•ҷзЁӢдёүзұ»ж•°жҚ®гҖӮжҜҸдёӘеӨ„зҗҶеҷЁж №жҚ®иҮӘиә«зү№зӮ№е®һзҺ°дәҶжәҗ/зӣ®ж Үи·Ҝеҫ„зҡ„иҺ·еҸ–гҖҒж–Үд»¶зӯӣйҖүгҖҒеӨ„зҗҶеҮҪж•°зҡ„йҖүжӢ©зӯүз»ҶиҠӮгҖӮ
    • дҫӢеҰӮпјҢForumQAProcessor е’Ң ForumTutorialProcessor йңҖиҰҒд»Һ combined.json ж–Үд»¶дёӯиҜ»еҸ–еҺҹе§Ӣж•°жҚ®пјҢе№¶ж №жҚ®ж—¶й—ҙжҲізӯүжқЎд»¶иҝҮж»ӨеҮәйңҖиҰҒеӨ„зҗҶзҡ„жқЎзӣ®пјҢйҒҝе…ҚйҮҚеӨҚе’Ңж— ж•ҲеӨ„зҗҶгҖӮ

    4. е№¶еҸ‘еӨ„зҗҶжңәеҲ¶

    • ж–Үд»¶еӨ„зҗҶйҮҮз”ЁеӨҡзәҝзЁӢе№¶еҸ‘жЁЎејҸгҖӮ_process_files ж–№жі•дјҡж №жҚ®й…ҚзҪ®зҡ„е№¶еҸ‘ж•°пјҢе°Ҷеҫ…еӨ„зҗҶж–Үд»¶еҲ—иЎЁеқҮеҲҶдёәиӢҘе№ІеӯҗеҲ—иЎЁпјҢжҜҸдёӘеӯҗеҲ—иЎЁз”ұдёҖдёӘзәҝзЁӢзӢ¬з«ӢеӨ„зҗҶгҖӮ
    • иҝҷз§Қи®ҫи®Ўе……еҲҶеҲ©з”ЁдәҶеӨҡж ё CPU иө„жәҗпјҢжҳҫи‘—жҸҗеҚҮдәҶеӨ§жү№йҮҸж–Үд»¶еӨ„зҗҶзҡ„йҖҹеәҰпјҢе°Өе…¶йҖӮеҗҲ I/O еҜҶйӣҶеһӢд»»еҠЎгҖӮ

    5. зҒөжҙ»зҡ„еӨ„зҗҶеҮҪж•°дёҺжөҒзЁӢеҲҶеҸ‘

    • жҜҸдёӘеӨ„зҗҶеҷЁйҖҡиҝҮ get_processing_function ж–№жі•пјҢеҠЁжҖҒйҖүжӢ©е…·дҪ“зҡ„ ETL жөҒзЁӢеҮҪж•°пјҲеҰӮ etl_doc_embedding_flowгҖҒetl_forum_qa_embedding_flow зӯүпјүпјҢе®һзҺ°дәҶжөҒзЁӢзҡ„зҒөжҙ»еҲҮжҚўе’Ңи§ЈиҖҰгҖӮ
    • иҝҷз§Қи®ҫи®ЎдҫҝдәҺеҗҺз»ӯжү©еұ•ж–°зҡ„еӨ„зҗҶжөҒзЁӢжҲ–ж–ҮжЎЈзұ»еһӢпјҢеҸӘйңҖж–°еўһеӯҗзұ»е’ҢеҜ№еә”зҡ„еӨ„зҗҶеҮҪж•°еҚіеҸҜгҖӮ

    6. ж–Үд»¶еҺ»йҮҚдёҺеўһйҮҸеӨ„зҗҶ

    • еңЁиҺ·еҸ–еҫ…еӨ„зҗҶж–Үд»¶еҲ—иЎЁж—¶пјҢзі»з»ҹдјҡиҮӘеҠЁеҜ№жҜ”жәҗзӣ®еҪ•е’Ңзӣ®ж Үзӣ®еҪ•пјҢзЎ®дҝқеҸӘеӨ„зҗҶе°ҡжңӘеӨ„зҗҶиҝҮзҡ„ж–Үд»¶пјҢйҒҝе…ҚйҮҚеӨҚи®Ўз®—гҖӮ
    • еҜ№дәҺи®әеқӣзұ»ж•°жҚ®пјҢиҝҳдјҡж №жҚ®ж—¶й—ҙжҲіиҝҮж»ӨжҺүиҝҮж—©жҲ–иҝҮж–°зҡ„ж•°жҚ®пјҢдҝқиҜҒж•°жҚ®зҡ„ж—¶ж•ҲжҖ§е’ҢеҮҶзЎ®жҖ§гҖӮ

    7. з»ҹдёҖе…ҘеҸЈдёҺжү№йҮҸи°ғеәҰ

    • etl_index_start еҮҪж•°дҪңдёәз»ҹдёҖе…ҘеҸЈпјҢиҙҹиҙЈж №жҚ®ж–ҮжЎЈзұ»еһӢе’Ңдә§е“ҒзәҝеҲқе§ӢеҢ–й…ҚзҪ®гҖҒйҖүжӢ©еӨ„зҗҶеҷЁе№¶еҗҜеҠЁеӨ„зҗҶжөҒзЁӢгҖӮ
    • еңЁ __main__ е…ҘеҸЈдёӯпјҢзі»з»ҹдјҡж №жҚ®з”ЁжҲ·иҫ“е…Ҙзҡ„е№¶еҸ‘ж•°пјҢиҮӘеҠЁеҜ№жүҖжңүдә§е“Ғзәҝе’Ңж–ҮжЎЈзұ»еһӢиҝӣиЎҢжү№йҮҸеӨ„зҗҶпјҢе®һзҺ°дәҶе…ЁиҮӘеҠЁеҢ–зҡ„ж•°жҚ® ETL и°ғеәҰгҖӮ

    и®ҫи®ЎеҺҹзҗҶдёҺе·ҘзЁӢиҖғйҮҸ

    1. йқўеҗ‘еҜ№иұЎдёҺй«ҳеҶ…иҒҡдҪҺиҖҰеҗҲ

    • йҖҡиҝҮжҠҪиұЎеҹәзұ»е’ҢеӨҡжҖҒеӯҗзұ»пјҢе°ҶдёҚеҗҢж–ҮжЎЈзұ»еһӢзҡ„еӨ„зҗҶйҖ»иҫ‘и§ЈиҖҰпјҢдҫҝдәҺз»ҙжҠӨе’Ңжү©еұ•гҖӮ
    • е…¬е…ұжөҒзЁӢпјҲеҰӮе№¶еҸ‘еӨ„зҗҶгҖҒж–Үд»¶еҺ»йҮҚпјүйӣҶдёӯеңЁеҹәзұ»е®һзҺ°пјҢеҮҸе°‘йҮҚеӨҚд»Јз ҒгҖӮ

    2. е№¶еҸ‘дёҺжҖ§иғҪдјҳеҢ–

    • еӨҡзәҝзЁӢеӨ„зҗҶжһҒеӨ§жҸҗеҚҮдәҶеӨ§и§„жЁЎж•°жҚ®еӨ„зҗҶзҡ„ж•ҲзҺҮпјҢе°Өе…¶йҖӮеҗҲж–Үд»¶ I/O еҜҶйӣҶеһӢеңәжҷҜгҖӮ
    • еҲ—иЎЁеҲҶеүІз®—жі•дҝқиҜҒдәҶеҗ„зәҝзЁӢиҙҹиҪҪеқҮиЎЎпјҢйҒҝе…Қиө„жәҗжөӘиҙ№гҖӮ

    3. еҒҘеЈ®жҖ§дёҺе®№й”ҷ

    • ж—Ҙеҝ—зі»з»ҹиҙҜз©ҝе…ЁжөҒзЁӢпјҢдҫҝдәҺй—®йўҳе®ҡдҪҚе’ҢиҝҪиёӘгҖӮ
    • еҗ„еӨ„зҗҶзҺҜиҠӮеқҮжңүејӮеёёжҚ•иҺ·пјҢеҚ•дёӘж–Үд»¶еҮәй”ҷдёҚдјҡеҪұе“Қж•ҙдҪ“жөҒзЁӢгҖӮ

    4. зҒөжҙ»зҡ„й…ҚзҪ®дёҺиҮӘеҠЁеҢ–

    • йҖҡиҝҮй…ҚзҪ®зұ»е’Ңз»ҹдёҖе…ҘеҸЈпјҢж”ҜжҢҒзҒөжҙ»и°ғж•ҙж №и·Ҝеҫ„гҖҒдә§е“ҒзәҝгҖҒе№¶еҸ‘ж•°е’ҢеӨ„зҗҶжЁЎејҸгҖӮ
    • ж”ҜжҢҒжү№йҮҸиҮӘеҠЁеҢ–еӨ„зҗҶпјҢжһҒеӨ§еҮҸе°‘дәҶдәәе·Ҙе№Ійў„е’Ңиҝҗз»ҙжҲҗжң¬гҖӮ
